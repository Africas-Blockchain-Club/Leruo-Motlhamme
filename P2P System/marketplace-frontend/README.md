# LM Marketplace — P2P Escrow Frontend

**Live site:** [https://lm-marketplace.netlify.app/](https://lm-marketplace.netlify.app/)

A peer-to-peer stablecoin escrow marketplace built with React, Vite, ethers.js, and GSAP.

## Tech Stack

- React 19 + Vite
- ethers.js v6
- GSAP animations
- Tailwind CSS
- Deployed on Netlify

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.
